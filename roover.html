Roover dispone de sistemas operativos:
Roover 20: 100€p : Este sistema operativo dispone de 4GB de RAM, y 200GB de almacenamiento, de las que el sistema ocupa 150.
Tiene Roover search y visor de archivos horizontales instaladas, además de correo, administrador de archivos, ajustes y Roover store. Capacidad para 8 aplicaciones. No se hacen actualizaciones y es vulnerable a algunos (no todos solo los más avanzados)(virus quantumm) virus.
Es necesario instalar Roover cuenta y crear una por 15€p. Las app de Roover store son más caras.
Roover 21 PRO: 200€p : Este sistema operativo tiene dispone 8GB de RAM y 676GB de almacenamiento de las que el sistema operativo ocupa 443.
Tiene Google, Roover maps, Word, adminisrador de archivos, ajustes, correo, Meet, Google drive, Roover store, Roover cuenta, Visor de archivos horizontales, Roover search, Roover documents, Roover Drive, Roover antivirus.
En ordenador fijo + Hojas de calculo y presentaciones
Tiene espacio para 20 aplicaciones y puede ampliarse con actualizaciones. Estas lo hacen menos vulnerables a virus.
Las aplicaciones nuevas las debes instalar tú.
La cuenta de Roover viene creada. Las app de Roover store son más baratas.
Si usted dispone de Roover 20 puede comprar únicamente el sistema operativo Rover 21 más barato.
Roover 21-P es un sistema operativo que cuesta 50€p exclusivo para pantallas donde se pueden colocar: dibujos, fotos, roover pending...
Roover 21-R
El almacenamiento viene determinado por la anchura de la pantalla del reloj. La RAM es suficiente. No se pueden instalar más apps que las pre instaladas o las que vengan con actualizaciones.
Roover 21-BOT
Este se usa para los robots, la RAM es la suficiente para usar el robot y el almacenamiento puede incluso determinarse por las actualizaciones pero es suficiente.
ROOVER 21-MOBILE
Usa las versiones de basic, medium advanced y pro (más información en el documento de información extra sobre sistemas operativos) y sirve para tablets y móviles, se acompaña de una sujección para poder poner el sistema operativo. Compatible con teléfono.
ROOVER 21-A
Este lo usan los asistentes virtuales, la RAM y el espacio es suficiente y se puede incrementar con actualizaciones o tarjetas SD.
